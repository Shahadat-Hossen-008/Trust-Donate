Important FAQ

What Is The Document Object Model(DOM)?

DOM convert our HTML document to an Object model. The DOM organizes elements in a hierarchical structure, making it easy to navigate and manipulate.

How do we select an element from DOM?

We can select an element from DOm five ways. Using getElementById:Selects an element by its ID. Using getElementsByClassName: Selects all elements with a specific class name. This returns a live HTMLCollection. Using getElementsByTagName: Selects all elements with a specific tag name. Also returns a live HTMLCollection. We don't often use that. Using querySelector: Selects the first element that matches a specified CSS selector. Using querySelectorAll: Selects all elements that match a specified CSS selector. This returns a static NodeList.

What is event delegation in the context of the DOM and why useful it?

When an event occurs on a child element, it bubbles up through the DOM tree to the parent. By using delegation reduces memory usage and improves performance by minimizing the number of event listeners.

How do we manipulate an elements attribute and styles by using the DOM?

We can manipulate an element's attributes and styles using the DOM in JavaScript through various methods and properties. By using setAttribute: we can set an attribute using the setAttribute method. By using getAttribute: To get the value of an attribute, we can use getAttribute. By using removeAttribute: To remove an attribute, use removeAttribute.
